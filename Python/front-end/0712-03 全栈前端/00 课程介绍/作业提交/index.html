    作业内容¶

    下载课件压缩包并解压出来之后，课件资料就存档在目录 0711-01-01000000 当中的 python 文件中，在里面将作业写完之后并且压缩提交就可以了。

    作业提交格式¶

    课程中每天都会有作业，作业提交记录与提交结果请查看作业提交表

    使用源代码的方式提交，题目用注释的方式写在源代码里面。

    • 作业文件命名：期次（0711）-01（第几次作业 二位数）-学号-姓名.zip（例如**0711-01-01000000-正心.zip**）

    • 前面需要加0711（07为全栈课程编号，11为全栈第11期）

    • 中间的内容，第一次为 01、第二次为 02， 依次递推
    • 第三个为学号，每个VIP学员都会有一个唯一的学号

    • 第四个给自己的名字，防止一些学员学号给错了统计不上信息

    • 提交到QQ邮箱：2328074219@qq.com

    • 作业在第二次课课前讲解，答案与第二次课代码一起发放。

    • 写作业遇到问题请微信联系，写完之后再QQ邮箱提交。
